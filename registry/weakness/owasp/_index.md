# OWASP (Weakness Namespace)

OWASP produces multiple weakness taxonomies for different domains, from traditional web security to AI/ML systems.

## Why OWASP Matters for Weaknesses

OWASP's Top 10 lists are the most widely recognized weakness taxonomies:

- **Practitioner-focused** - Written for developers and security teams
- **Regularly updated** - Tracks evolving threat landscape
- **Community-driven** - Open contributions and feedback
- **Free and open** - No licensing restrictions

## Weakness Sources in This Namespace

| Name | Description | Example ID |
|------|-------------|------------|
| `top10` | Web Application Security Top 10 | A01, A10 |
| `llm-top10` | LLM Application Security Top 10 | LLM01, LLM10 |
| `ml-top10` | Machine Learning Security Top 10 | ML01, ML10 |
| `ai-exchange` | Comprehensive AI Security Framework | PROMPTINJECTION |
| `agentic-top10` | Agentic AI Security Top 10 | AGT01 |
